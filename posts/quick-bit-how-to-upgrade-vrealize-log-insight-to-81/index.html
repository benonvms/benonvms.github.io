Quick Bit - How to Upgrade vRealize Log Insight 4.7 to 8.1

Unfortunatly, it is not possible to upgrade from vRealize Log Insight from 4.7.1 to 8.1. So we have to do an intermidiate step first.

The Cluster:

  • Master Node: 192.168.201.91
  • Second Node: 192.168.201.92
  • Third Node: 192.168.201.93
  • Cluster VIP: 192.168.201.90

The Preperation

Download the following files from my.vmware.com:

  • VMware-vRealize-Log-Insight-4.8.0-13036238.pak
  • VMware-vRealize-Log-Insight-8.1.0-15994158.pak

After that we need about an hour of downtime1 to perform the upgrade.

The Upgrade:

  • First we connect to the master node, and select Administrator / Cluster and pick Upgrade Cluster.
  • After that we have to select the VMware-vRealize-Log-Insight-4.8.0-13036238.pak file.
  • The upgrade will begin to upgrade the Master Node first and then roll thorough the cluster.
  • After all nodes upgraded sucessfully to Version 4.8.0 we can perform the same steps to upgrade to Log Insight 8.1

  1. At least this is the time it took in my lab. ↩︎
