Quick Bit - My vCenter is not able to talk to my AD

Sometimes the vCenter Server is not able to talk to the AD. And all Likewise commands, like

/opt/likewise/bin/domainjoin-cli query

Fail with an error like

Error: LW_ERROR_ERRNO_ECONNREFUSED [code 0x00009d49]

This is happens when the Likewise Service Control Manager Service is not running.

This can be checked with the following command.

root@vcsa-01 [ ~ ]# systemctl status lwsmd.service

If the Service is not running you can try the following

root@vcsa-01 [ ~ ]# systemctl enable --now lwsmd.service

This will make the service boot persistent again and starts it automatically.
