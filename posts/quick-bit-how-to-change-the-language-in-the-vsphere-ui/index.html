Quick Bit - How to Change the Language in the vSphere UI?

Sometimes you need to change the display language of the vSphere Clients.

HTML 5 UI (vCenter Server 6.7+)

  • Login to the vSphere Client
  • Click on you username
  • Select “My Preferences”
  • In the Tab Language you can specify a language.

vSphere Host Client (vSphere 6.5+)

For the Host Client it is similar

  • Select your user
  • Select Settings, Language and then your preferred Language
